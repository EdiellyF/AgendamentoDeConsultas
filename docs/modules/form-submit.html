Módulo de Agendamento

Arquivo fonte: src/modules/form/submit.js

Este módulo é responsável por gerenciar o formulário de agendamento de consultas. Ele permite que os usuários registrem novos agendamentos com informações do paciente, incluindo nome, CPF e horário da consulta.

Funcionalidades Principais

  • Captura de dados do paciente (nome e CPF)
  • Validação de campos obrigatórios
  • Seleção de data e hora da consulta
  • Geração automática de ID único para cada agendamento
  • Limpeza automática do formulário após o agendamento

Campos do Formulário

  • Nome do Paciente: Campo obrigatório para identificação do paciente
  • CPF: Documento de identificação do paciente (formatação automática)
  • Data: Data da consulta (limitada à data atual em diante)
  • Horário: Seleção do horário disponível para a consulta

Observações Importantes

  • A data mínima permitida é sempre a data atual
  • É obrigatório selecionar um horário para realizar o agendamento
  • O CPF é automaticamente formatado (remove pontos e traços)
  • Após o agendamento, os campos são limpos automaticamente

Fluxo de Funcionamento

  1. Usuário preenche os dados do paciente
  2. Seleciona a data desejada
  3. Escolhe um horário disponível
  4. Sistema valida os dados inseridos
  5. Realiza o agendamento se todos os dados estiverem corretos
  6. Atualiza a lista de horários disponíveis
  7. Limpa o formulário para um novo agendamento

Tratamento de Erros

  • Alerta se o nome do paciente não for preenchido
  • Alerta se nenhum horário for selecionado
  • Log de erro no console em caso de falha no agendamento

Dependências

- dayjs: Para manipulação de datas
- scheduleNew: Serviço para criar novo agendamento
- schedulesday: Função para atualizar a lista de horários
- gerarUIDD: Serviço para gerar IDs únicos
