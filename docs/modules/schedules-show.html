Schedule Show

Módulo responsável pela exibição visual dos agendamentos na interface

Arquivo fonte: src/modules/schedules/show.js
Arquivo fonte: src/modules/schedules/show.js

Visão Geral

O módulo show.js é responsável por renderizar os agendamentos na interface do usuário, organizando-os em períodos (manhã, tarde e noite) e criando elementos DOM dinâmicos para cada agendamento.

Dependências

dayjs

Biblioteca para manipulação de datas

import dayjs from "dayjs";

Métodos

showSchedules({ dailySchedules })

Função principal que renderiza os agendamentos na interface.

Parâmetros:

dailySchedules: Array de objetos contendo os agendamentos do dia

Comportamento:

  • Valida se o parâmetro é um array
  • Limpa agendamentos anteriores
  • Cria elementos DOM para cada agendamento
  • Distribui os agendamentos nos períodos corretos

Tratamento de Erros:

  • Valida tipo do parâmetro dailySchedules
  • Exibe alerta em caso de erro
  • Registra erros no console

Elementos DOM

Seletores Utilizados:

  • #period-morning: Container de agendamentos da manhã
  • #period-afternoon: Container de agendamentos da tarde
  • #period-night: Container de agendamentos da noite

Elementos Criados Dinamicamente:

<li data-id="[id]">
    <strong>[horário]</strong>
    <span>[nome]</span>
    <span>[cpf]</span>
    <img class="cancel-icon" src="./src/assets/cancel.svg" alt="Cancelar">
</li>

Estrutura do Agendamento

Objeto de Agendamento:

{
    id: string,
    name: string,
    cpf: string,
    when: string // formato: "YYYY-MM-DD HH:mm:ss"
}

Distribuição por Período:

  • Manhã: horários até 12h
  • Tarde: horários entre 12h e 18h
  • Noite: horários após 18h

⚠️ Importante

  • Certifique-se que os elementos de período existem no DOM
  • O formato de data deve ser compatível com dayjs
  • Todos os campos do objeto de agendamento são obrigatórios

Exemplo de Uso

import { showSchedules } from './modules/schedules/show.js';

            

                
Implementação

                
// Array de agendamentos
const dailySchedules = [
    {
        id: "123",
        name: "João Silva",
        cpf: "123.456.789-00",
        when: "2024-03-20 10:30:00"
    }
];

// Renderiza os agendamentos
showSchedules({ dailySchedules });
