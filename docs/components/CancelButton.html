CancelButton

Módulo responsável pela funcionalidade de cancelamento de agendamentos

Visão Geral

O módulo cancelButton.js implementa a funcionalidade de cancelamento de agendamentos no sistema. Ele gerencia a interação do usuário com o botão de cancelamento e realiza as operações necessárias para remover o agendamento tanto da interface quanto do banco de dados.

Funcionalidade

Event Listener

O módulo adiciona um event listener para cada período de agendamento (.period) que:

  • Detecta cliques no ícone de cancelamento
  • Solicita confirmação do usuário
  • Remove o agendamento do banco de dados
  • Atualiza a interface removendo o item da lista

confirmarRegistroDeletado()

Função que exibe um diálogo de confirmação para o usuário.

Retorno:

Boolean: true se o usuário confirmar, false caso contrário

Dependências

  • scheduleDeleteById: Serviço para deletar agendamento por ID
  • schedulesday: Função para atualizar a visualização dos agendamentos

Exemplo de Uso

O módulo é carregado automaticamente e adiciona a funcionalidade de cancelamento aos ícones:

<li data-id="123">
    <strong>14:00</strong>
    <span>Nome do Paciente</span>
    <img src="cancel.svg" class="cancel-icon" alt="Cancelar" />
</li>
Nota: Certifique-se de que os elementos HTML possuam as classes e data-attributes corretos para o funcionamento adequado do módulo.
