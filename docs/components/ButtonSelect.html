ButtonSelect

Classe para gerenciamento de seleção de botões em formulários

Visão Geral

A classe ButtonSelect é responsável por gerenciar a seleção de botões em um formulário, permitindo alternar classes CSS entre elementos baseado em datasets.

Propriedades Privadas

  • #element: Elemento DOM principal

Construtor

new ButtonSelect(element)

Cria uma nova instância de ButtonSelect.

Parâmetros:

element (string | Element): Seletor CSS ou elemento DOM

Throws:

Lança um erro se o elemento não for encontrado

Métodos

selectElement(classeElement, datasetIDElement, nameDataSet)

Adiciona um evento de clique que alterna uma classe CSS entre elementos.

Parâmetros:

classeElement (string): Classe CSS a ser alternada

datasetIDElement (string): ID do dataset do elemento alvo

nameDataSet (string): Nome do atributo dataset a ser utilizado

getElement()

Retorna o elemento DOM principal.

Retorno:

Element: Elemento DOM

Exemplo de Uso

// Criando uma instância
const buttonSelect = new ButtonSelect('#meuBotao');
// Configurando a seleção do elemento
buttonSelect.selectElement(
   'botao-ativo',           // Classe CSS a ser alternada
   'botao1',                // ID do dataset
   'data-botao'             // Nome do dataset
;
